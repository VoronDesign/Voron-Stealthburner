# Filename Nomenclature

| Filename | Note |
|----------|------|
| \*.stl | Base color |
| [a]\_\*.stl | Accent color |
| [o]\_\*.stl | Opaque (light doesn't penetrate) |
| [c]\_\*.stl | Clear/Translucent (light can penetrate) |
| \*\_cw1.stl | Clockwork1 Version |
| \*\_cw2.stl | Clockwork2 Version |
